# Mini RPG Arena
A text-based game

## Version 0.1
## Description
A simple text-based RPG game written in Python where the player fights a Goblin.

### Features
- Fight a Goblin
- Basic HP System
- Terminal Based

### Technologies
- Python


## Version 0.2
### Features
- Turn-based combat system
- Player VS Goblin battle
- Random damage system (between 3-7)
- HP system for both player and enemy
- Win/Lose conditions
- Simple menu system

## How it works
- Player chooses an action (Fight or Exit)
- If Fight
    - Player attacks enemy
    - Enemy HP decreases
    - Enemy attacks back
    - Player HP decreases
    - Game ends whenever HP of either side reaches 0

### Technologies
- Python

### How to Run
```bash
python main.py
